Danh sách API - Dating APP

1. Đăng ký tài khoản

TYPE: POST

URL: https://dating-deploy.vapor.cloud/users

BODY PARAMS:
 email  email
 password mật khẩu
 name tên
Example Params: { "name": "Nam", "email": "user2@gmail.com", "password": "12345" }

2. Login

TYPE: POST

Authorization Type: Basic Auth

URL: https://dating-deploy.vapor.cloud/login



3. Xem user hiện tại ( get curent user )

TYPE: GET

Authorization Type: Bear Token ( token lấy từ API Login )

URL: https://dating-deploy.vapor.cloud/me
